    This project showcases comparisons of different weather patterns for cities across the globe, specifically at cities' latitudes. The following weather patterns were examined: temperature (in Fahrenheit), humidity, cloudiness, and wind speed (mph). Throughout this site, you can navigate to data visualizations as well as examine the data itself. There are global comparisons, as well as comparisons for the age-old debate: which hemisphere has better weather: the northern hemisphere (positive latitudes) or the southern hemisphere (negative latitudes)? Enjoy the findings, and hope you learn some interesting facts about weather trends across the globe!
